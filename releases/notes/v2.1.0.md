## Highlights

- **HxDBMigrate** — schema analysis, case-type generation, polling sync with
  health score, one-click cutover with 72 h rollback window, RS256-signed
  compliance certificate.
- **HxMeet** — embedded video/screen-share (self-hosted LiveKit) or
  Teams/Zoom/Meet connectors; single-use guest invites; consent-gated sealed
  recordings & transcripts (tenant-DEK encrypted, audit-chain anchored); GPU
  live captions (Whisper); AI post-session summaries; document-first KYC with
  liveness challenges and Art. 9 biometric consent.
- **HxReplay + costing** — counterfactual single/cohort replay, rate cards,
  manual timers/timesheets, billing export (JSON/CSV).
- **HxEvolve** — mines process drift, proposes replay-proven or
  evidence-labelled changes, human-approved via HxBranch; cumulative-drift
  freeze guardrail.
- **HxNexus Operator (MCP)** — JSON-RPC MCP surface at `/api/v1/mcp`; read
  tools by default, opt-in writes with durable idempotency, human-confirm
  proposals, and scoped external-agent tokens with instant revocation.
- **HxNexus case Q&A** — cited answers over case variables, timeline,
  messages, documents, and transcripts; sovereignty-first (local models by
  default, external providers only by explicit tenant consent).
- **Marketplace execution layers** — L1 inert-connector installs with
  subset-only grants, host allowlists and SSRF guards; L2 digest-pinned
  hardened containers on an internal egress-blocked network with a scoped
  read-only broker; drift detection on both tiers.
- **Portal v2** — offline-capable PWA with queued submissions and dedupe,
  worker↔customer messaging, CSAT, customer workflow steps, logo upload.
- **HxDraft** — draft-first configuration changes with diff cards, checksum
  conflict detection, and human merge.
