# Velaris v1.0.0 — Initial Release

**Released:** 2026-06-04  
**Type:** Initial release

---

## Case Management

- Create, assign, and manage cases with full lifecycle tracking
- Visual Case Designer — drag-and-drop stage and step builder
- Custom case types with configurable variables
- Case locking — prevent concurrent edits
- Case numbers — human-readable reference IDs
- Soft delete with full audit trail
- Sub-cases — link related cases (returns, complaints, escalations)
- Case canvas (HxCanvas) — freeform whiteboard per case

---

## Work Center

- Agent workspace for processing case steps
- Step-by-step task completion with validation
- Inline action panels — payments, KYC, e-sign, SMS, documents
- Live activity feed per case
- Case timeline — full chronological event log

---

## Form Builder

- Visual drag-and-drop form designer
- Field types: text, number, date, dropdown, checkbox, file upload, signature
- Conditional logic — show/hide fields based on answers
- Form versioning and branching (HxBranch)

---

## Customer Portal

- Public-facing portal at `/portal/:slug` — no login required
- Case submission, status tracking, document upload
- SLA countdown ring per case
- Customer dashboard — view all cases by email
- Case timeline visible to customer
- AI chat (HxNexus) inline on portal
- Bank details OTP flow for secure financial data collection
- Branded per tenant — custom colour, logo, welcome text

---

## User & Access Management

- User Directory — invite, manage, deactivate users
- Access Groups — role-based access control (ABAC)
- Route-level permissions per role
- Superadmin account with breach detection
- Real authentication with JWT + SSO provider support
- Credential rotation with full audit log
- Session revocation

---

## AI — HxNexus

- Natural language case queries
- AI-assisted case step suggestions
- Public portal AI chat
- NLP builder for custom AI prompts
- Scout AI — intelligent case routing and triage

---

| Integration | Capability |
|-------------|-----------|
| Stripe | Payments, refunds, checkout sessions |

| S3 / MinIO | Cloud file storage routing |

- Connector intelligence — AI-assisted connector configuration

## HxBridge — Connector Engine

- Unified connector execution layer
- REST, webhook, and event-driven connector types
- Dev connector (DevConn) for testing integrations locally

---

## Document Management

- Upload, store, and attach documents to cases
- OCR extraction via Docling
- Shared documents visible to customers on the portal
- MinIO-backed storage with signed URL access

---

## Email

- Full SMTP email integration
- Email templates per case event
- Inbound email → case creation
- Mailpit for local dev email testing

---

## Push Notifications

- Web push notifications per case event
- VAPID-based push admin
- FCM support

---

## Analytics — HxAnalytics

- Case volume and throughput dashboards
- SLA breach tracking and alerts
- Process mining — discover bottlenecks in case workflows
- Agent performance metrics

---

## Compliance & Audit

- Tamper-proof audit log (7-year retention)
- SOC 2 / GDPR / HIPAA compliance toggles
- DML before-image logging
- Full event trail per case, user, and connector

---

## Security — HxShield

- Real-time threat detection
- Superadmin breach alerts
- Account auto-disable on suspicious activity
- Security event webhook (Slack, PagerDuty, SIEM)
- Internal IP range blocking

---

## Multi-Tenancy

- Full tenant isolation — data, users, settings, portal
- Per-tenant branding
- Tenant-level feature toggles

---

## Workflow & Orchestration

- Temporal-powered workflow engine
- Visual process orchestrator
- BPM importer — import workflows from 12 vendors (BPMN, Visio, Lucidchart, etc.)
- HxFusion — cross-case workflow automation
- Escalation trees with SLA-based triggers

---

## Developer & Admin Tools

- HxMigrate — AI-assisted data migration from external systems
- HxDeploy — CI/CD pipeline for Velaris configuration (dev → UAT → prod)
- HxBranch — Git-like versioning for case types, forms, connectors
- HxDocs — in-platform documentation builder
- HxSync — real-time data sync between Velaris and external systems
- HxGlobal — multi-language and locale support
- HxGraph — live knowledge graph of the entire platform
- HxStream — real-time event streaming
- Observability — logs, metrics, health dashboard

- Docker 24+
- 4 GB RAM minimum (8 GB recommended)
- 10 GB disk minimum (without Ollama AI)
- 20 GB disk minimum (with Ollama AI)
